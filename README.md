# RickAndMortyApp - Android Programming Exam

**Course:** Android Programming (15 ECTS)   
**Institution:** Kristiania University College    
**Tech:** Kotlin, Jetpack Compose, Android Studio  

RickAndMortyApp is an app built around the [Rick and Morty API](https://rickandmortyapi.com/). The case covers fetching and displaying characters from the API, letting users create and save their own custom characters to a local database, and building a custom fourth screen tied to the Rick and Morty theme.
